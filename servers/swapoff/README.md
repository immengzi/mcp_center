# 交换空间停用MCP（管理控制程序）规范文档
## 一、服务介绍
本服务是一款基于`swapoff`命令实现交换空间停用的MCP（管理控制程序），核心功能为​​停用交换空间（Swap）​​，释放已启用的交换分区或交换文件​​，将其从系统内存管理中移除。

## 二、核心工具信息
| 工具名称 | 工具功能 | 核心输入参数 | 关键返回内容 |
| ---- | ---- | ---- | ---- |
| `swapoff_disabling_swap_tool` | ​停用交换空间（Swap）​​，释放已启用的交换分区或交换文件​​，将其从系统内存管理中移除 | - `host`：远程主机名/IP（本地采集可不填）<br>- `name`：停用的swap空间路径 | 布尔值，表示停用指定swap空间是否成功 |
