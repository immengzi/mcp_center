# Echo文本写入MCP（管理控制程序）规范文档

## 一、服务介绍
本服务是一款基于`echo`命令实现文本写入功能的MCP（管理控制程序），核心功能为将指定文本内容写入到文件中。支持本地和远程主机文件写入，可以使用各种echo选项（如不输出换行符等）进行高级文本写入操作，并支持覆盖写入和追加写入两种模式。

## 二、核心工具信息
| 工具名称 | 工具功能 | 核心输入参数 | 关键返回内容 |
| ---- | ---- | ---- | ---- |
| `echo_write_to_file_tool` | 使用echo命令将文本写入文件 | - `host`：远程主机名/IP（本地操作可不填）<br>- `text`：要写入的文本内容<br>- `file`：要写入的文件路径<br>- `options`：echo选项（可选），如"-n"不输出换行符等<br>- `mode`：写入模式，"w"表示覆盖写入，"a"表示追加写入，默认为"w" | 布尔值，表示写入操作是否成功 |
