# 系统内存整体状态信息采集MCP（管理控制程序）规范文档
## 一、服务介绍
本服务是一款基于`free`命令实现系统内存整体状态信息采集的MCP（管理控制程序），核心功能为精准收集目标设备的物理内存信息，包括系统内存总量、系统已使用内存、空闲的物理内存、系统可分配给新应用程序的内存量。

## 二、核心工具信息
| 工具名称 | 工具功能 | 核心输入参数 | 关键返回内容 |
| ---- | ---- | ---- | ---- |
| `free_collect_tool` | 获取目标设备（本地/远程）中内存整体状态信息 | - `host`：远程主机名/IP（本地采集可不填） | 内存信息列表（含`total`系统内存总量（MB）、`used`系统已使用内存量(MB)、`free`空闲物理内存（MB）、`available`系统可分配内存（MB）） |
