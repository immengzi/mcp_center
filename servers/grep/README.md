# 文件内容搜索MCP（管理控制程序）规范文档

## 一、服务介绍
本服务是一款基于`grep`命令实现文件内容搜索的MCP（管理控制程序），核心功能为在指定文件中搜索包含特定模式的行。支持本地和远程主机文件搜索，可以使用各种grep选项（如忽略大小写、显示行号等）进行高级搜索。

## 二、核心工具信息
| 工具名称 | 工具功能 | 核心输入参数 | 关键返回内容 |
| ---- | ---- | ---- | ---- |
| `grep_search_tool` | 在文件中搜索指定模式的内容 | - `host`：远程主机名/IP（本地搜索可不填）<br>- `options`：grep选项（可选），如"-i"忽略大小写，"-n"显示行号等<br>- `pattern`：要搜索的模式（支持正则表达式）<br>- `file`：要搜索的文件路径 | 包含匹配行的字符串，如果没有找到匹配项则返回相应的提示信息 |
