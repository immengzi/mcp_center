| 工具名称                 | 工具功能                                                 | 核心输入参数                                                  | 关键返回内容                                                                                                                 |
| -------------------- | ---------------------------------------------------- | ------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| `hotspot_trace_tool` | 使用 `perf record` 和 `perf report` 分析系统或指定进程的 CPU 性能瓶颈 | - `host`：远程主机名/IP（可选，不填则分析本机）、`pid`：目标进程 ID（可选，不填则分析整机） | - `total_samples`：总样本数<br>- `event_count`：事件计数（如 cycles）<br>- `hot_functions`：热点函数列表（按 Children 百分比排序，包含函数名、库、符号类型和占比） |
