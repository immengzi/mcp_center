| 工具名称             | 工具功能            | 核心输入参数                                                               | 关键返回内容                                                                                                                                     |
| ---------------- | --------------- | -------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| `strace_syscall` | 采集指定进程的系统调用统计信息 | - `host`：可选，远程主机地址<br>- `pid`：目标进程ID（必填）<br>- `timeout`：采集超时时间，默认10秒 | List\[Dict]，每个字典包含：<br>- `syscall`：系统调用名称<br>- `total_time`：总耗时（秒）<br>- `call_count`：调用次数<br>- `avg_time`：平均耗时（微秒）<br>- `error_count`：错误次数 |
